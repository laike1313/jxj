# 物品审批系统

一个基于HTML、CSS和JavaScript开发的物品审批管理系统。系统提供了用户登录、申请提交、审批管理等功能。

## 功能特点

- **用户登录/注册系统**：支持用户注册和登录功能，包括密码找回
- **申请管理**：用户可以提交物品申请，上传附件，查看申请状态
- **审批流程**：管理员可以查看、批准或拒绝申请
- **数据统计**：图形化展示申请类型和审批状态
- **个人设置**：用户可以修改个人信息和系统设置

## 技术实现

- 前端：HTML5、CSS3、JavaScript
- 存储：浏览器本地存储 (LocalStorage)
- 图表：自定义CSS实现的数据可视化
- 响应式设计：适配不同尺寸的屏幕
